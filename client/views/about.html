Quiz4All is a simple application which allows users to have fun, learn or test themselves by taking a QUIZ!!!

  • Quiz4All is implemented both as Web and Hybrid Mobile application. That’s why users can use the app in convenient way on any device.
  • Quiz4All in Web version allows a user to register and login.
  • Registered user is able to create a new quiz.
  • To create a quiz a user needs to enter quiz name and description (optionally can also upload quiz image).
  • New quiz will have at least one question (each question has to have at least one answer).
  • Quiz4All in Hybrid Mobile version only allows to take a quiz – To create a quiz user needs to use web application.
